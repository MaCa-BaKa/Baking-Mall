# 21-烘焙商城管理系统
21-烘焙商城管理系统    技术： 基于 B/S 架构 SpringBoot+MySQL+Vue+layui  环境： Idea mysql maven jdk1.8


## 软件功能

管理端主要功能描述
（1）用户管理
（2）店铺管理 :增删改查店铺
（3）烘焙分类管理 :对产品进行类别分类
（4）烘焙信息管理 

（5）烘焙订单管理 :对所有的订单进行管理

（6）烘焙评价管理

（7）系统管理

用户端主要功能描述

（1）店铺展示
（2）烘焙资讯
（3）收藏功能
（4）烘焙信息管理:点击购买后将商品提交到购物车

（5）烘焙订单管理 :对自己已下订单进行支付 查询状态

（6）烘焙评价管理

（7）购物车:对已添加商品进行数量 控制进行支付下单
